#  AI Proctored Exam System – Frontend

This project is the frontend module of an AI-based Proctored Examination System designed to monitor candidates during online exams and prevent malpractice using real-time camera monitoring and automated detection techniques.

The application provides a secure, user-friendly interface for students to attend exams and for administrators to supervise exam activities efficiently.

---

##  Features

-  Secure Login Authentication
-  Live Webcam Monitoring during exam
-  Face Detection & Activity Tracking
-  Malpractice Alerts (multiple faces / no face / tab switch)
-  Online Test Interface
-  Timer-based exam submission
-  Admin Dashboard for monitoring
-  Responsive UI

---

##  Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Web APIs (Camera / MediaDevices)
- REST API integration (Backend)
